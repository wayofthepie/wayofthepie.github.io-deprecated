Leetcode: Check if N and its double exists

2021-02-01 leetcode

Quick post about a solution to the leetcode problem Check If N and Its Double Exist.

pub fn check_if_exist(arr: Vec<i32>) -> bool {
    arr.iter().enumerate().any(|(i, n)| {
        arr[..i]
            .iter()
            .any(|&elem| ((n % 2 == 0) && (elem == n / 2)) || elem == (n * 2))
    })
}

This is looking at each element in the Vec of numbers. We iterate through the Vec getting both the value and index of the current number. Then, we compare all previously seen numbers before our current number by either multiplying or dividing the current number by 2. If n is the current number the rules are:

  • if n is divisible by 2, compare the previously seen number (elem in the code above) with n / 2
  • if that is not true, or the number does not match, compare it to n * 2

If neither match, continue on to the next previously seen number until we reach the index of our current number. Then we know our current number has no doubles in the Vec and also that nothing doulbes to it. If we ever get a match, any is short-circuiting, so we return the first (if any) match in both iterations.
